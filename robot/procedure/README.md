# Рефлексия

## Плюсы

Код приобрел модульный стиль: функции парсинга отделены от непосредственной реализации так, что при необходимости внесения изменений это можно делать изолировано на отдельных функциях: "Start", "Set" и т.д.

## Минусы

Неудобства вызывает большое количество ветвлений, которые в коде заменил свитч-кейсами и которые в большом проекте будет крайне сложно поддерживать.

Завязка на глобальное состояние.
